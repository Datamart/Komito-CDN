How to integrate Komito Analytics using Google Tag Manager

Finally, a quick and simple guide on how to integrate Komito Analytics and gain additional insights about your visitors behavior.

 

There are two simple and easy ways to integrate Komito Analytics. The first one, is to simply import the prepared container template, and second, is to copy and paste the short code into a Custom HTML Tag.

 

1. Import Container

Please follow these simple steps to import GTM container from JSON file:
  • Download the container template file.
  • Sign in to Google Tag Manager.
  • Select an existing container or create a new one.
  • Go to the Admin section.
  • Under Container options select Import Container.
  • Preview and Publish.

 

For more information on importing containers, please see Google Tag Manager Help Center.

 

2. Create Custom HTML Tag

Please follow these simple steps to create a Custom HTML tag:
  • Sign in to Google Tag Manager.
  • Select an existing container or create a new one.
  • Click Tags and then New.
  • Click Tag Configuration and select Custom HTML.
  • Copy the tag code and paste it into the HTML field: 

<script src="https://komito.net/komito.js" async></script>
<script>
var _komito = _komito || {
  'trackTwitter': 1,    // Tracks Twitter events if widget is presented on page.
  'trackFacebook': 1,   // Tracks Facebook events if widget is presented on page.
  'trackLinkedIn': 1,   // Tracks LinkedIn events if plugin is presented on page.
  'trackDownloads': 1,  // Tracks files download links.
  'trackOutbound': 1,   // Tracks outbound links.
  'trackForms': 1,      // Tracks forms submissions.
  'trackUsers': 1,      // Tracks pageviews by users logged in to social networks.
  'trackActions': 1,    // Tracks 'mailto', 'tel', 'sms' and 'skype' actions.
  'trackPrint': 1,      // Tracks page print actions.
  'trackMedia': 1,      // Tracks HTML5 video, audio, Vimeo and YouTube players events.
  'trackScroll': 1,     // Tracks scroll depth.
  'trackOrientation': 1,// Tracks orientation change on mobile devices.
  'trackAdblock': 0,    // Tracks page views with blocked ads. (Experimental)
  'trackErrorPages': 0, // Tracks error pages. (Experimental)
  'sendHeartbeat': 0,   // Sends heartbeat event. (Default interval 30 seconds)
  'debugMode': 0,       // Prints all requests to console.
  'nonInteraction': ['form', 'print', 'scroll', 'video', 'orientation', 'adblock'] // List of non interaction events.
};
</script>
  • Update the metrics that you want to track automatically by changing "1" to "0".
  • Select "All Pages" on Triggering section.
  • Save, Preview and Publish.

 

For more information about Custom HTML tags, please see Google Tag Manager Help Center.
